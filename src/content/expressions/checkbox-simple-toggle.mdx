## Usage Notes

### Setup

1. Add a **Checkbox Control** effect to your control layer
2. Name the effect something descriptive
3. Apply this expression to the **Opacity** property of the layer you want to toggle

### Customization

- Swap `0` and `100` to invert the behavior (checked = hidden)
- Use on other properties by changing the output values (e.g., `[0,0,0]` and `[100,100,100]` for scale)
