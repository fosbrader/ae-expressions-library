## Usage Notes

### Project Structure

This expression expects a specific project organization:

```
_CONTROL (comp)
├── Home TEAM CONTROL (layer with checkbox effects)
├── Away TEAM CONTROL (layer with checkbox effects)
└── ...

Home_Endcard (comp)
├── Player_Card (layer with this expression)
└── ...
```

### Naming Requirements

1. **Comp names:** `TeamName_Purpose` (e.g., `Home_Endcard`)
2. **Control layers:** `TeamName TEAM CONTROL` (e.g., `Home TEAM CONTROL`)
3. **Layer names:** `Part0_Part1` (e.g., `Player_Card`)
4. **Checkbox effects:** Match layer name pattern (e.g., `Player_Card`)

### When to Use

- Large template systems with multiple team/variant comps
- Centralized control panel for toggling elements across all comps
